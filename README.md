# LiarCard

LiarCard 是一款基于 Unity 开发的纸牌撒谎博弈游戏。

玩家需要通过出牌、撒谎、跟牌、质疑、测谎仪和职业技能，尽快打空自己的手牌。

## 核心特色

* 纸牌撒谎玩法
* Joker 万能牌机制
* 跟牌 / 不跟 / 质疑
* 测谎仪概率提示
* 职业技能系统
* 单机 AI 对战

## 基础玩法

游戏使用一副或多副扑克牌，包含大小王。

Joker 可以作为任意点数使用。

每轮由一名玩家开启，声明一个点数并打出任意数量的牌。
后续玩家只能围绕该点数进行跟牌，也可以选择不跟或质疑。

质疑时只检查上一名玩家刚出的牌：

* 如果全部符合声明点数或为 Joker，质疑失败。
* 如果存在不符合声明的普通牌，质疑成功。

最先出完手牌，并且没有被成功质疑的玩家获胜。

## 测谎仪机制

其他玩家出牌后，测谎仪会显示其说谎概率。

该概率不是绝对准确值，而是受以下因素影响：

* 出牌数量
* 职业特性
* 技能效果
* 历史质疑结果
* 随机误差

## 职业系统

初版计划设计 4 个职业：

| 职业  | 定位 | 效果              |
| --- | -- | --------------- |
| 侦探  | 推理 | 提高测谎准确度         |
| 骗术师 | 伪装 | 降低自己被测出的说谎概率    |
| 赌徒  | 风险 | 强化质疑收益和失败惩罚     |
| 魔术师 | 变化 | 可将普通牌临时视为 Joker |

## 技术架构

项目采用分层架构：

```txt
Assets/Scripts/
├── Core/
├── Gameplay/
├── Systems/
│   ├── LieDetector/
│   ├── Skill/
│   └── AI/
├── Presentation/
│   ├── UI/
│   └── View/
├── Data/
└── Infrastructure/
```

### 架构说明

* `Core`：核心规则和数据
* `Gameplay`：游戏流程控制
* `Systems`：测谎仪、职业技能、AI
* `Presentation`：UI 和视觉表现
* `Data`：ScriptableObject 配置
* `Infrastructure`：音效、存档、事件等基础服务
